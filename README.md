# Carts SPA

Современное одностраничное приложение для работы с корзинами покупок

## Технологии

- **React** 18  
- **TypeScript**  
- **TanStack Query** v5 (ранее React Query)  
- **React Router**  
- **Emotion** (styled-components подход)  
- **React Toastify** — уведомления  
- **Lodash.debounce** — защита от спама запросов  

## Основной функционал

- Список всех корзин с пагинацией  
- Skeleton-loader во время загрузки данных  
- Переход на страницу детального просмотра корзины  
- Просмотр списка товаров в корзине  
- Изменение количества каждого товара  
- Toast-уведомления при:  
  - успешном изменении количества  
  - ошибках мутаций  
  - ошибках запросов  

## Важный момент про API

> Приложение работает с **имитационным API**, которое **не сохраняет** изменения на сервере.  
> Все действия (изменение количества, удаление и т.д.) **не отображаются клиенту**.
