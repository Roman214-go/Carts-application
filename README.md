## Технологии

- **React** 18
- **TypeScript**
- **TanStack Query** v5
- **React Router**
- **Emotion**
- **React Toastify**
- **Lodash.debounce**

## Основной функционал

- Список всех корзин с пагинацией
- Skeleton-loader во время загрузки данных
- Переход на страницу детального просмотра корзины
- Просмотр списка товаров в корзине
- Изменение количества каждого товара
- Toast-уведомления при:
  - успешном изменении количества
  - ошибках мутаций
  - ошибках запросов

## Важный момент про API

> Приложение работает с **имитационным API**, которое **не сохраняет** изменения на сервере.  
> Все действия (изменение количества, удаление и т.д.) **не отображаются клиенту**.

## Архитектура проекта

Для данного приложения была создана довольно простая архитектура из-за его размера: без разделения на модули, выноса в константы переменных, **smart** и **dumb** компонентов и т.п.

```text
src/
├── pages/          # страницы приложения
├── components/     # UI-компоненты
├── hooks/          # кастомные хуки
├── store/          # глобальное состояние
├── types/          # общие типы
└── service/        # API-запросы

```
