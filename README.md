### Carts SPA

## Технологии

React 18
TypeScript
TanStack Query v5
React Router
Emotion (styled)
React Toastify
Lodash.debounce

## Функциональность

Список корзин
Пагинация
Skeleton-loader во время загрузки
Переход на страницу деталей корзины
Просмотр списка товаров
Изменение количества товара
Toast-уведомления при успешных и ошибочных действиях
Toast-уведомления при ошибках мутаций
UI/UX

### Важный момент про API

Приложение использует API, которое имитирует обновление данных.
Сервер не сохраняет изменения корзины.
Все обновления (изменение количества, удаление товара) не отображаются клиенту.

## Архитектура

pages/ - страницы приложения
components/ - переиспользуемые UI-компоненты
hooks/ - кастомные хуки (React Query)
store/ - глобальное состояние (пагинация)
types/ - типизация данных
service/ - запросы
